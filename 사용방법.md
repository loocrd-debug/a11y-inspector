# 가디언즈 오브 겔럭시 v2.0
## 웹 접근성 · 오탈자 · 링크 종합 검사 도구

---

## 📁 이 폴더 안에 있는 파일

```
가디언즈-오브-겔럭시/
  ├── 처음설치.bat     ← ① 처음 한 번만 실행 (Windows)
  ├── 실행.bat         ← ② 매번 이걸로 실행  (Windows)
  ├── 실행.sh          ← Mac/Linux 실행용
  ├── server.cjs       ← 서버 프로그램 (건드리지 마세요)
  ├── package.json     ← 라이브러리 목록 (건드리지 마세요)
  ├── data/
  │   └── minwon-list.xlsx  ← 민원 목록 엑셀 (8,461건)
  └── public/
      ├── index.html         ← 메인 화면
      └── minwon.html        ← 민원 검사 화면
```

---

## 🖥️ 사용 방법 (Windows)

### ① Node.js 설치 (처음 1회)

1. https://nodejs.org 접속
2. 왼쪽 초록 버튼 **"LTS"** 클릭 → 다운로드
3. 설치 파일 실행 → 계속 **"다음"** 클릭

### ② 처음설치.bat 실행 (처음 1회)

**[처음설치.bat]** 더블클릭 → 자동으로 설치 완료

> 인터넷 연결 필요 / 약 5~10분 소요

### ③ 실행.bat 으로 시작

**[실행.bat]** 더블클릭 → 서버 시작 + 브라우저 자동 열림

→ 브라우저에서 http://localhost:3000 확인

---

## 🏛️ 민원 검사 방법

1. http://localhost:3000/minwon.html 접속
2. **"배치 스캔 실행"** 탭 클릭
3. **"전환 차수별 검사"** 섹션에서 원하는 버튼 클릭
   - 1차 (1,963건) / 2차 (3,237건) / 3차 (3,261건)
4. 확인 → 검사 시작!

---

## ⚠️ 주의사항

- **이 창(검은 창)을 닫으면 서버가 종료됩니다**
- 검사 중에는 창을 닫지 마세요
- 전체 검사는 수 시간이 걸릴 수 있습니다
- 인터넷에 연결되어 있어야 합니다

---

## ❓ 문제 해결

| 증상 | 해결 방법 |
|------|-----------|
| "node를 찾을 수 없음" | Node.js 재설치 후 PC 재시작 |
| 브라우저가 안 열림 | http://localhost:3000 직접 입력 |
| 설치 중 오류 | 인터넷 연결 확인 후 [처음설치.bat] 재실행 |
| 포트 오류 | 기존 창 닫고 [실행.bat] 재실행 |
